aCatuai.pde
-----------

Version 0.03
------------

This program descends from Jim Gallt aCatuai and his utility sketches

It combines the calibration with aCatuai for the user interface 
and temperature logging functions.  Addition simplistic low level of test
of the ADC,  ambient temperature sensor and eeprom where added. 

This program is intended be controlled via the serial monitor screen
without a LCD so you can probe and troubleshoot TC4 shields.
